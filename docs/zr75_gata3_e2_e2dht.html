The quantro test (Hicks and Irizarry 2015) for both equality of medians and equality of distributions was applied to logCPM values without normalisation. Distributions were found to have equal medians (ANOVA p = 0.561).

When checking for equality of distributions 1000 permutations were performed, acknowledging that for datasets with two groups and 6 samples, this far exceeds the number of viable permutations. For these small datasets, however, a permutation p-value < 0.1 is still possible and should be noted with more weight than one otherwise might.

After performing this step, the quantro statistics obtained from the set of permutations were subset to only the unique values, assuming that each permutation will return a distinct statistic. This subset of permutations was then used to define the effective number of permutations and the exact p-value was calculated using common strategies (as described in Phipson and Smyth 2010), by adding one to the number of tests greater that the observed test statistic, and likewise adding one to the number of effective permutations. This returned a permutation p-value of 0.273 suggesting that RLE normalisation is appropriate.

Alignments were counted for the set of sliding windows using windows of 180bp shifting in increments of 60bp. Windows were required to return a total of at least 12 alignments to be included during counting.

The complete set of sliding windows was then reduced using dualFilter() from extraChIPs with thresholds defined to return ~70% of sliding windows which overlap a consensus peak. As a result ranges were retained from the top 99.16% signal percentiles, where signal was beyond 1.33 times the level of signal in the input samples. This retained a total of 381,504 windows from the original set of 21,581,109 windows across the entire genome.

Quantro Test

The quantro test (Hicks and Irizarry 2015) for both equality of medians and equality of distributions was again applied to logCPM values from sliding windows without normalisation. Distributions were found to have equal medians (ANOVA p = 0.564).

Using the same strategy for permutation testing as above, a permutation p-value of 0.364 was returned suggesting that RLE normalisation is appropriate. Importantly SQ-normalisation (Hicks et al. 2018) is expected to be robust to any results produced by this test.
